HOW TO USE:
═══════════════════════════════════════════════════════════════════════════

FOR USERS:
  1. Click the Sun/Moon icon in the top-right (desktop)
  2. Or find Theme toggle in mobile menu
  3. Theme switches instantly
  4. Preference is automatically saved

FOR DEVELOPERS:
  
  Method 1 - CSS Variables (Recommended):
    <div className="bg-background text-foreground">
      Auto-adapts to theme
    </div>
  
  Method 2 - useTheme Hook:
    const { theme, toggleTheme } = useTheme()
  
  Method 3 - Tailwind dark: prefix:
    <div className="bg-white dark:bg-slate-900">
      Light and dark mode
    </div>
